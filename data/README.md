Transliteration files to English as a target language used for Sequitur testing. 
File format: one word per line: 
word (in source language) [white space] its transliteration (in target language)

E.g.:

kirenius quirinius
kasdime chaldea
yunus jonas
andronikus andronicus
sardis sardis
agripa agrippa
salamis salamis
pilatus pilate
lisanias lysanias
kayapas caiaphas ... 

Or:

구레뇨가 quirinius
갈대아를 chaldea
요한의 jonas
사데 sardis
아그립바왕이 agrippa
살라미에 salamis

아빌레네 lysanias
가야바의 caiaphas
람은 aram
달마누다 dalmanutha
골로새에 colossae
...
